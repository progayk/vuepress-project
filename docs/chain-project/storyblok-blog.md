## Create the Project

### Create Nuxt App

Go to terminal:
* create a nuxt app.
* Say `Yes` to `Universal` and `ESLint` and select `Axios`.

```bash
create-nuxt-app <project_name>
```

Run development server to preview 

```bash
npm run dev
```
